| №      | Пункт      | Простыми словами                                                                                                                                                                                 | Обычно ставят                     |
| ------ | ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ | --------------------------------- |
| **0**  | **SQL**    | Убирает шипение, когда никто не говорит. Чем больше число, тем сильнее должен быть сигнал, чтобы рация его услышала. Если поставить слишком много — можешь не услышать товарища, который далеко. | **2–3**                           |
| **1**  | **STEP**   | На сколько изменяется частота при вращении или вводе вручную. Если не настраиваешь частоты руками — можно не трогать.                                                                            | **12.5 kHz**                      |
| **2**  | **TXP**    | Мощность передачи. HIGH — дальше "добивает", но быстрее садит аккумулятор. LOW — экономит батарею и обычно хватает на небольших дистанциях.                                                      | **LOW / HIGH по ситуации**        |
| **3**  | **SAVE**   | Экономия батареи. Пока никто не говорит, рация "дремлет" и расходует меньше энергии.                                                                                                             | **ON**                            |
| **4**  | **VOX**    | Передача голосом без нажатия кнопки PTT. Любой громкий звук может случайно включить передачу.                                                                                                    | **OFF**                           |
| **5**  | **WN**     | Ширина канала связи. Если выбрать неправильно, голос может стать тихим или искажённым.                                                                                                           | **NARROW**                        |
| **6**  | **ABR**    | Через сколько секунд погаснет подсветка экрана после последнего нажатия кнопки.                                                                                                                  | **5 сек**                         |
| **7**  | **TDR**    | Одновременно слушает две частоты (верхнюю и нижнюю строку на экране).                                                                                                                            | **ON или OFF**                    |
| **8**  | **BEEP**   | Писк при нажатии кнопок.                                                                                                                                                                         | **OFF**                           |
| **9**  | **TOT**    | Максимальное время одной передачи. Если случайно зажмешь кнопку — рация сама остановит передачу.                                                                                                 | **60–90 сек**                     |
| **10** | **R-DCS**  | DCS-код, который должен прийти, чтобы рация открыла динамик. Используется только если вся группа работает с DCS.                                                                                 | Обычно **OFF**                    |
| **11** | **R-CTCS** | То же самое, только для CTCSS-тонов.                                                                                                                                                             | Обычно **OFF**                    |
| **12** | **T-DCS**  | DCS-код, который твоя рация будет отправлять при передаче.                                                                                                                                       | По настройкам группы              |
| **13** | **T-CTCS** | CTCSS-тон, который отправляется при передаче.                                                                                                                                                    | По настройкам группы              |
| **14** | **VOICE**  | Голосовые подсказки ("Channel mode", "Frequency mode").                                                                                                                                          | **OFF**                           |
| **15** | **ANI-ID** | Отправляет цифровой идентификатор рации при передаче. Для обычного использования не нужен.                                                                                                       | **OFF**                           |
| **16** | **DTMFST** | Определяет, будут ли слышны DTMF-сигналы при нажатии цифровых клавиш.                                                                                                                            | Не менять                         |
| **17** | **S-CODE** | Выбор заранее записанного DTMF-кода. Используется редко.                                                                                                                                         | Не менять                         |
| **18** | **SC-REV** | Как работает сканирование: ждать окончания передачи или сразу продолжать поиск.                                                                                                                  | **TO**                            |
| **19** | **PTT-ID** | Когда отправлять ANI-ID: до передачи, после или никогда.                                                                                                                                         | **OFF**                           |
| **20** | **PTT-LT** | Задержка перед отправкой ANI-ID.                                                                                                                                                                 | По умолчанию                      |
| **21** | **MDF-A**  | Что показывать в верхней строке: частоту, название канала или номер.                                                                                                                             | **NAME** или **FREQ**             |
| **22** | **MDF-B**  | То же самое для нижней строки.                                                                                                                                                                   | **NAME** или **FREQ**             |
| **23** | **BCL**    | Если кто-то уже говорит на этой частоте — не даст начать передачу и перебить его.                                                                                                                | **ON**                            |
| **24** | **AUTOLK** | Автоматически блокирует клавиатуру через некоторое время.                                                                                                                                        | По желанию                        |
| **25** | **SFT-D**  | Используется только при работе через ретранслятор (смещение частоты).                                                                                                                            | **OFF**                           |
| **26** | **OFFSET** | Размер смещения частоты для ретрансляторов.                                                                                                                                                      | Не менять                         |
| **27** | **MEM-CH** | Сохранить текущую частоту в память рации.                                                                                                                                                        | По необходимости                  |
| **28** | **DEL-CH** | Удалить сохранённый канал из памяти.                                                                                                                                                             | По необходимости                  |
| **29** | **WT-LED** | Цвет подсветки, когда рация ничего не делает.                                                                                                                                                    | Любой                             |
| **30** | **RX-LED** | Цвет подсветки во время приёма сигнала.                                                                                                                                                          | Любой                             |
| **31** | **TX-LED** | Цвет подсветки во время передачи.                                                                                                                                                                | Любой                             |
| **32** | **AL-MOD** | Настройки тревожной сигнализации (Alarm). В обычной работе не используется.                                                                                                                      | По умолчанию                      |
| **33** | **BAND**   | Выбор диапазона работы: VHF или UHF.                                                                                                                                                             | По используемой частоте           |
| **34** | **TDR-AB** | Какая строка (верхняя или нижняя) будет главной при двойном прослушивании.                                                                                                                       | **A**                             |
| **35** | **STE**    | Убирает неприятный шум ("хвост") после окончания передачи.                                                                                                                                       | **ON**                            |
| **36** | **RP-STE** | Такая же функция, но только для работы через ретрансляторы.                                                                                                                                      | **OFF**                           |
| **37** | **RPT-RL** | Задержка работы RP-STE.                                                                                                                                                                          | По умолчанию                      |
| **38** | **PONMSG** | Что показывать при включении рации: приветствие или служебную информацию.                                                                                                                        | По вкусу                          |
| **39** | **ROGER**  | После окончания передачи звучит короткий "бип". Многие считают его лишним.                                                                                                                       | **OFF**                           |
| **40** | **RESET**  | Сбросить настройки рации. Есть частичный и полный сброс. Использовать только при необходимости.                                                                                                  | Не использовать без необходимости |


CTCSS = аналоговый тон-пароль
DCS = цифровой пароль (более устойчивый)
